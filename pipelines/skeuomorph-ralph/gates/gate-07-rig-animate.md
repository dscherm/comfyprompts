# Quality Gate 7: RIG-ANIMATE

## PASS Criteria

### Props (skipped path)
- [ ] Stage 7 is marked `"skipped"` with `gate_passed: true` in `pipeline-state.json`
- [ ] `output/rigged/rigged-model.glb` exists (copied from validated)
- [ ] `output/animated/static-model.glb` exists (copied from validated)

### Characters (humanoid)
- [ ] `output/rigged/rigged-model.glb` exists and is >10KB
- [ ] Armature is present in the GLB with at least 19 bones
- [ ] Weight coverage is >85% (at least 85% of vertices have at least one bone weight >0.01)
- [ ] At least idle animation exists and plays (non-zero frame range)
- [ ] `output/animated/combined.glb` exists with at least 1 named animation track
- [ ] `rig-report.json` and `animation-report.json` exist
- [ ] Materials and textures are preserved in all output GLBs

### Creatures
- [ ] `output/rigged/rigged-model.glb` exists and is >10KB
- [ ] Armature is present with at least 1 spine chain and root bone
- [ ] Weight coverage is >85%
- [ ] At least idle animation exists and plays
- [ ] `output/animated/combined.glb` exists with at least 1 named animation track
- [ ] `rig-report.json` and `animation-report.json` exist
- [ ] Materials and textures are preserved in all output GLBs

## WARN Criteria (log but don't block)
- [ ] Bone count is below typical range for the asset type (minimal rig but functional)
- [ ] Weight coverage is between 85-90% (passing but slightly below ideal)
- [ ] Run animation is missing (idle and walk present for characters)
- [ ] Finger bones absent (simplified hand rig)
- [ ] Loop continuity warning -- walk/run cycle has slight pose discontinuity at loop point
- [ ] Rig was generated by fallback method (Blender Rigify or cloud instead of UniRig)

## FAIL Criteria (block advancement -- re-run Stage 7)
- [ ] Rigged GLB does not exist or is corrupt
- [ ] No armature found in rigged GLB (rigging failed)
- [ ] Bone count is 0
- [ ] Weight coverage is <50% (most vertices unweighted -- mesh will collapse in animation)
- [ ] No animation clips produced (animate step failed)
- [ ] Any animation clip has NaN transforms in keyframes (will cause mesh to disappear)
- [ ] Materials were lost during rig export (0 material slots in rigged GLB)
- [ ] Combined GLB failed to export (individual clips exist but merge step failed)

### Armature and bone count check
```bash
"C:/Program Files/Blender Foundation/Blender 5.0/blender.exe" \
  --background --python - <<'PYTHON' -- RIGGED_GLB
import bpy, sys

argv = sys.argv[sys.argv.index("--") + 1:]
glb_path = argv[0]

bpy.ops.object.select_all(action='SELECT')
bpy.ops.object.delete()
bpy.ops.import_scene.gltf(filepath=glb_path)

armatures = [obj for obj in bpy.data.objects if obj.type == 'ARMATURE']
meshes = [obj for obj in bpy.data.objects if obj.type == 'MESH']

if not armatures:
    print("FAIL: No armature found in rigged GLB")
else:
    for arm in armatures:
        bone_count = len(arm.data.bones)
        bone_names = [b.name for b in arm.data.bones]
        print(f"Armature: {arm.name}, bones: {bone_count}")
        print(f"  Bones: {bone_names[:10]}{'...' if len(bone_names) > 10 else ''}")
        if bone_count < 5:
            print(f"  FAIL: Bone count {bone_count} is suspiciously low")
        elif bone_count >= 19:
            print(f"  PASS: Bone count meets minimum (19+)")
        else:
            print(f"  WARN: Bone count {bone_count} is below typical minimum (19)")

for mesh in meshes:
    mat_count = len([s for s in mesh.material_slots if s.material])
    print(f"Mesh {mesh.name}: {mat_count} materials preserved")
    if mat_count == 0:
        print(f"  FAIL: No materials on {mesh.name}")
PYTHON
```

### Weight coverage check
```bash
"C:/Program Files/Blender Foundation/Blender 5.0/blender.exe" \
  --background --python - <<'PYTHON' -- RIGGED_GLB
import bpy, sys

argv = sys.argv[sys.argv.index("--") + 1:]
glb_path = argv[0]

bpy.ops.object.select_all(action='SELECT')
bpy.ops.object.delete()
bpy.ops.import_scene.gltf(filepath=glb_path)

for obj in bpy.data.objects:
    if obj.type != 'MESH':
        continue
    total_verts = len(obj.data.vertices)
    covered = sum(
        1 for v in obj.data.vertices
        if any(g.weight > 0.01 for g in v.groups)
    )
    pct = covered / total_verts * 100 if total_verts else 0
    status = "PASS" if pct >= 85 else ("WARN" if pct >= 50 else "FAIL")
    print(f"{obj.name}: {pct:.1f}% weight coverage ({status})")
PYTHON
```

### Animation track check
```bash
"C:/Program Files/Blender Foundation/Blender 5.0/blender.exe" \
  --background --python - <<'PYTHON' -- COMBINED_GLB
import bpy, sys

argv = sys.argv[sys.argv.index("--") + 1:]
glb_path = argv[0]

bpy.ops.object.select_all(action='SELECT')
bpy.ops.object.delete()
bpy.ops.import_scene.gltf(filepath=glb_path)

actions = bpy.data.actions
if not actions:
    print("FAIL: No animation clips found in combined GLB")
else:
    print(f"Animation clips found: {len(actions)}")
    for action in actions:
        frames = int(action.frame_range[1] - action.frame_range[0])
        has_nan = any(
            kp.co[1] != kp.co[1]  # NaN check
            for fcurve in action.fcurves
            for kp in fcurve.keyframe_points
        )
        status = "FAIL (NaN keyframes)" if has_nan else "PASS"
        print(f"  {action.name}: {frames} frames -- {status}")
PYTHON
```
